# 4) Програмний код інтерпретатора операторного виконання (п.8)

Файл програмного коду:

- `lab12/extarithexpr-ass.py`

Що реалізовано відповідно до п.8:

- Виконання програми як послідовності операторів.
- Оператор присвоєння: `name = arith_expr`.
- Оператор виведення: `print(arith_expr)`.
- Таблиця змінних (словник `name -> value`).
- Використання змінних у формулах.
- Підтримка всіх розширень виразів з п.5-6 (`//`, `%`, `**`, `sin`, `cos`, унарні знаки, дійсні числа).

Коментарі в коді:

- У файлі додані коментарі до граматики, сканера, виконавця операторів, розбору виразів і тестових прикладів.
