## Тестова програма 1

```python
x = 10
y = 3.5
z = x * y - 2 ** 3
print(z)
print(x // y)
print(x % 3)
w = sin(3.14159265358979 / 2)
print(w)
```

### Отриманий результат

```text
27.0
2.0
1
1.0
```

### Коментар

- Результат збігається з виконанням стандартного Python.

## Тестова програма 2

```python
a = 5
b = a ** 2
c = -a + b
print(c)
d = cos(0) + sin(0)
print(d)
e = (a + 3) // 2
f = (a + 3) % 2
print(e)
print(f)
result = a ** 3 - 3 * a ** 2 + 3 * a - 1
print(result)
```

### Отриманий результат

```text
20
1.0
4
0
64
```

### Коментар

- Результат збігається з виконанням стандартного Python.
- Показано роботу присвоєння, `print()`, змінних, унарних операцій, степеня, `//`, `%`, тригонометричних функцій.
