# 2) Тестові приклади формул для перевірки інтерпретатора (п.6)

Набір тестів достатній для демонстрації всіх елементів п.6.

## Базовий тест з умови

- `49 - 108 / (6 + 11) * (100 - 94)`

## Операції `//` і `%` (п.5.1)

- `17 // 5`
- `17 % 5`
- `17 // 5 + 17 % 5`
- `(10 + 7) // 3`

## Піднесення до степеня `**` (п.5.2)

- `2 ** 10`
- `2 * 3 ** 2`
- `2 ** 3 ** 2`
- `8 ** (1 / 3)`

## Функції `sin()` і `cos()` (п.5.3)

- `sin(0)`
- `cos(0)`
- `sin(3.14159265/2)`
- `cos(3.14159265)`
- `2*sin(1)+cos(0)`

## Унарні `+` і `-` (п.5.4)

- `-2 + -6`
- `4 * +8`
- `--5`
- `---5`
- `-2 ** 2`
- `-(10 - 3)`
- `-sin(0)`

## Дійсні числа та експоненційна форма (п.5.5)

- `45.02 + 0.98`
- `1.0e-5 * 100000`
- `28e4 / 2`
- `2.5E-2 + 0.1`

## Комплексні вирази

- `sin(2) ** 2 + cos(2) ** 2`
- `-3 ** 2 + 17 // 5 - 4 % 3`

## Тести помилок

- `5 / 0`
- `5 // 0`
- `2 + * 3`
- `log(10)`
- `(2 + 3`
